# KLK Group · Copilot Cowork Immersion

A static GitHub Pages site for an internal Copilot immersion session for KLK Group, focused on **Microsoft 365 Copilot Cowork** for licensed users.

🔗 **Live site:** https://steffiyappy.github.io/klk-cowork-immersion/
🔐 **Access code:** `PlantationCowork10May`

## What's inside

- A password-gated single-page HTML app (`index.html`) with:
  - Overview, Story, Get Started
  - Exercise 1 — Q1 Yield Analysis (Excel + Analyst agent)
  - Exercise 2 — Operations Reports (Word + PowerPoint)
  - **Exercise 3 — Cowork ★ (centerpiece)** — five parallel Board-prep deliverables in one prompt
  - Exercise 4 — Agent Builder (Plantation Advisor)
  - More Plantation Use Cases
  - Prompting Tips (GCSE framework + plantation prompt library)
- Demo files in `files/`:
  - `01_Contoso_YieldData.xlsx` — Q1 plantation yield data
  - `02_Contoso_OpsReport.docx` — Q1 Group Operations Report

## Persona & scenario

The site uses a fictional persona — **Lim Siew Ling, Group COO at Contoso Plantation Berhad** — preparing for her quarterly Board Operations Review. All numbers, estates and people are fictional.

## Audience

For M365 Copilot licensed users at KLK Group. The session is delivered as a guided demonstration.
